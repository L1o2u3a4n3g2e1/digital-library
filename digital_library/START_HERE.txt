🚀 QUICK START (3 STEPS - 5 MINUTES)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Open Command Prompt
   Windows: Press Win+R → type "cmd" → press Enter
   Mac:     Press Cmd+Space → type "terminal" → press Enter
   Linux:   Open your terminal application

STEP 2: Navigate to project and install dependencies
   Copy & paste this into your terminal:

   cd C:\Users\PC\Desktop\digital_library && npm install

   Then wait for installation to complete (takes 1-2 minutes)

STEP 3: Start the server
   Copy & paste this into your terminal:

   npm start

   You should see this message:
   ✓ Digital Library Server running on http://localhost:5000
   ✓ Connected to in-memory SQLite database
   ✓ Database initialized with sample data

   Now open your browser and go to: http://localhost:5000

🎉 That's it! The system is running!

⚙️ CREATE PACKAGE.JSON FILE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

If package.json doesn't exist, create a new file with this content:

{
  "name": "digital-library-system",
  "version": "1.0.0",
  "description": "Multilingual Digital Library System",
  "main": "server.js",
  "scripts": {
    "start": "node server.js",
    "dev": "node server.js"
  },
  "dependencies": {
    "express": "^4.18.2",
    "sqlite3": "^5.1.6",
    "body-parser": "^1.20.2",
    "cors": "^2.8.5",
    "bcryptjs": "^2.4.3",
    "jsonwebtoken": "^9.0.0"
  }
}

1. Open Notepad or your text editor
2. Paste the above content
3. Save as "package.json" in the digital-library folder
4. Run: npm install

🔧 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ "npm: command not found"
   → Download Node.js from https://nodejs.org/
   → Restart your terminal after installing

❌ "Port 5000 already in use"
   → Edit server.js, change: const PORT = 5000 → const PORT = 5001
   → Access at: http://localhost:5001

❌ "Voice search not working"
   → Use Chrome, Edge, or Safari (best voice support)
   → Check microphone permissions in browser settings
   → Try speaking clearly in a quiet environment

❌ "Text-to-speech not working"
   → Most modern browsers support this
   → Check system volume is not muted
   → Try a shorter text first

❌ "Can't connect to localhost:5000"
   → Check if server is running: npm start
   → Check for error messages in terminal
   → Try: http://127.0.0.1:5000

Want to understand how the system works?

Frontend (React):
  • public/index.html contains all UI components
  • Uses React hooks (useState, useEffect)
  • Implements Translation system
  • Uses Web Speech API for TTS/STT

Backend (Node.js):
  • server.js is the main API server
  • Uses Express.js framework
  • SQLite3 for database
  • JWT for authentication
