📥 HOW TO DOWNLOAD EACH BOOK
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

METHOD 1: Download from Project Gutenberg Website (Easiest)

1. Visit: https://www.gutenberg.org/ebooks/[ID]
   Replace [ID] with the number (e.g., 1342 for Pride and Prejudice)

2. Scroll down to "Download Files" section

3. Look for "Plain Text UTF-8" option

4. Click on the .txt link
   Or use direct format: https://www.gutenberg.org/cache/epub/[ID]/pg[ID].txt

5. Browser will download the file
   (Usually goes to Downloads folder)

6. Move file to appropriate category folder:
   Windows: Cut and paste to books/[Category]/

---

METHOD 2: Direct Download Links (Fastest)

Just replace the ID number in this URL:
https://www.gutenberg.org/cache/epub/1342/pg1342.txt

Examples:
- Pride and Prejudice (1342): https://www.gutenberg.org/cache/epub/1342/pg1342.txt
- The Origin of Species (2009): https://www.gutenberg.org/cache/epub/2009/pg2009.txt
- David Copperfield (766): https://www.gutenberg.org/cache/epub/766/pg766.txt

---

METHOD 3: Using a Browser Extension

1. Install Project Gutenberg downloader extension
2. Visit book page
3. Click extension button
4. Choose format (TXT)
5. Save location

📝 FILE NAMING CONVENTION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Format: [Title]_[Author_LastName].txt

Examples:
✓ Pride_and_Prejudice_Austen.txt
✓ The_Origin_of_Species_Darwin.txt
✓ David_Copperfield_Dickens.txt
✓ Wealth_of_Nations_Smith.txt

Rules:
- Use underscores instead of spaces
- Use title case for titles
- Use author's last name only
- Keep filenames under 50 characters
- No special characters (!@#$%^&*)
