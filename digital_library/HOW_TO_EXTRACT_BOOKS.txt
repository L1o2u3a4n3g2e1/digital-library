TROUBLESHOOTING

Problem: "No books found in Agriculture folder"

          Check that you clicked the upload button before submitting.

Problem: "Access denied" error
Solution: Make sure the folder C:\Users\PC\Desktop\digital_library\ exists.
          You may need to run Command Prompt as Administrator.

Problem: PowerShell execution policy error
Solution: The script handles this automatically with the batch file.
          If running manually, that's the issue.

Problem: Files won't extract

          Try running the script again.
