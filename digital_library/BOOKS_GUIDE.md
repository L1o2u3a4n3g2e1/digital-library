## Where to Download FREE Books Legally

### 🔗 Best Sources for Free English Books

#### 1. **Project Gutenberg** (https://www.gutenberg.org)
- 70,000+ FREE books (public domain)
- All books are free to download
- Available in multiple formats (TXT, EPUB, HTML, etc.)
- No registration required
- Most reliable source for classics

**How to use:**
1. Go to https://www.gutenberg.org
2. Browse by category or search
3. Click on book title
4. Download in your preferred format

#### 2. **Standard Ebooks** (https://standardebooks.org)
- Carefully formatted public domain books
- Higher quality editions
- Free downloads
- Beautiful typography

#### 3. **Open Library** (https://openlibrary.org)
- Borrow or read free books
- Large collection
- Can search by category
- Some books available for instant reading

#### 4. **LibriVox** (https://librivox.org)
- Free audiobooks
- Public domain
- Search by category
- Download as MP3

#### 5. **Internet Archive** (https://archive.org)
- Massive collection of books
- Free borrowing and downloads
- Multiple formats available
- Good for various categories

#### 6. **Google Books** (https://books.google.com)
- Preview and read free books
- Search by category
- Download where available
- Large collection

## 📥 How to Download Books

### **Method 1: Download from Project Gutenberg**

1. Visit: https://www.gutenberg.org/ebooks/[BOOK_ID]
   - Replace [BOOK_ID] with the number (e.g., 1342 for Pride and Prejudice)

2. Scroll down to "Download Options"

3. Choose format:
   - **TXT** - Plain text (best for our system)
   - **EPUB** - For e-readers
   - **HTML** - For web reading
   - **Kindle** - For Kindle devices

4. Save to your computer

5. Move file to corresponding category folder:
   ```
   books/Agriculture/[filename].txt
   books/Health/[filename].txt
   books/Love/[filename].txt
   etc.
   ```

### **Method 2: Download as TXT Files**

Direct download links (TXT format):
- Just change format in URL to `.txt` version
- Example: https://www.gutenberg.org/cache/epub/1342/pg1342.txt

## 🆘 Troubleshooting

**Problem: Can't find book on Project Gutenberg**
→ Try searching by author name
→ Check alternative sources (Open Library, Standard Ebooks)

**Problem: File too large**
→ Use TXT format (smaller than other formats)
→ Remove extra blank lines if needed

**Problem: Encoding issues**
→ Ensure UTF-8 encoding
→ Use Notepad++ for viewing
→ Project Gutenberg provides multiple encodings
